# Sola Busca Tarot

Date added: `2026-05-15`

This directory contains the local 78-card pack used for the `sola-busca` deck.

## Source

- Commons category: `https://commons.wikimedia.org/wiki/Category:Sola-Busca_tarot_deck`
- The source category also contains extra non-core images; they are excluded from the curated pack.

## Coverage

- Full deck: `78/78`
- Minors and trumps are both connected
- Back side and extra category files are excluded from the pack

## License / reuse

- Commons file pages for the numbered card scans mark the images as public domain.
- Recheck the individual Commons file pages if the assets are reused outside this repository.

## Notes

- The pack is a historical Italian gaming tarot deck.
- The app meanings remain an educational layer on top of the imagery.
- The files are normalized to the app's canonical card ids so the UI can resolve them locally without runtime hotlinks.
