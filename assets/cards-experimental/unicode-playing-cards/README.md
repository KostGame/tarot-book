# Unicode Playing Cards

- Source reference: [Playing Cards (Unicode block)](https://en.wikipedia.org/wiki/Playing_Cards_(Unicode_block))
- Pack type: locally rendered SVG cards derived from Unicode playing-card characters
- Coverage: `38/38` major arcana + court cards only
- Mapping note: the 22 trumps map to `FOOL` + `TRUMP-1..21`; the four tarot suits map to the Unicode suit families (`SPADES`, `HEARTS`, `DIAMONDS`, `CLUBS`)
- Native titles: the UI shows the Unicode card names as a secondary line
- License / reuse: the assets are generated locally from Unicode characters and do not depend on external hotlinked images
- Added: 2026-05-18

This pack is intentionally limited to majors and court cards. Pips are not included.
