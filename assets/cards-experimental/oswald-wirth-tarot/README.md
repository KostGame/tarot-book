# Oswald Wirth Tarot

Local card images for the `Oswald Wirth` deck.

## Source

- Commons category: https://commons.wikimedia.org/wiki/Category:Oswald_Wirth_tarot_deck

## Pack notes

- The local pack contains 22 card images.
- Only the numbered BnF major-arcana scans are included.
- Duplicate and alternate scans from the Commons category are excluded.
- The files are renamed to the app's canonical major card ids so the UI can resolve them locally without runtime hotlinks.

## License / reuse

The Commons file pages for the numbered scans describe the images as public domain. Recheck the individual Commons file pages if the assets are reused outside this project.

## Usage in the app

This deck is treated as a historical major-only tarot deck from 1889. The app's card meanings remain an educational layer on top of the imagery.
