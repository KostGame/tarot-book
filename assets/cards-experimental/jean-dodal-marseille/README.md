# Jean Dodal Marseille Pack

Date added: `2026-05-11`

This directory contains the local 22-card major-only pack used for the `jean-dodal-marseille` deck.

## Source

- Commons category: `https://commons.wikimedia.org/wiki/Category:Tarot_de_Marseille_-_Jean_Dodal`
- Secondary source: `http://www.tarot-history.com/Jean-Dodal/`

## Coverage

- Major arcana only: `22/22`
- Minors are intentionally not included

## License / reuse

- The Commons scans are public-domain reproductions.
- Check the individual Commons file pages and the tarot-history.com credit if you reuse this pack outside the project.

## Notes

- The pack is mapped by title, not by blind numeric reuse.
- `Strength` uses the source scan titled `La Force`.
- `Justice` uses the source scan titled `La Justice`.
- The app shows the translated card title as the primary label and the French native title in parentheses.
