# Tarot Nouveau / Grimaud 1898

Local card images for the `Tarot Nouveau` deck.

## Source

- Commons category: https://commons.wikimedia.org/wiki/Category:Tarot_nouveau_-_Grimaud_-_1898
- Gallica / BnF source: http://gallica.bnf.fr/ark:/12148/btv1b10510159t

## Pack notes

- The local pack contains 78 card images.
- The back side image is excluded from the deck pack.
- The files are renamed to the app's canonical card ids so the UI can resolve them locally without runtime hotlinks.

## License / reuse

The Commons file pages describe the 1898 scans as public domain / PD-old / PDM. Recheck the individual Commons file pages if the assets are reused outside this project.

## Usage in the app

This deck is treated as a French gaming tarot deck from 1898. The app's card meanings remain an educational layer on top of the imagery.
